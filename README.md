## Deskripsi Project

Project ini merupakan program grafik komputer menggunakan Python,
Pygame, dan PyOpenGL yang menampilkan:

-   Objek Kubus 3D
-   Objek Persegi 2D

Program mendukung transformasi: - Translasi - Rotasi - Skala - Shearing
(2D) - Refleksi (2D)

------------------------------------------------------------------------

## Library Yang Digunakan

-   pygame
-   PyOpenGL

------------------------------------------------------------------------

## Cara Menjalankan Program

### 1. Install Library

``` bash
pip install pygame PyOpenGL PyOpenGL_accelerate
```

### 2. Jalankan Program

``` bash
python main.py
```
